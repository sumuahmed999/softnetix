## ðŸ”§ Customization

### Change Colors

Edit the CSS variables in `admin/dashboard.php`:

```css
/* Primary gradient */
background: linear-gradient(135deg, #4FD1C5, #63B3ED);

/* Status colors */
.stat-card.new {
    --card-color: #3B82F6;
    --card-bg: #EFF6FF;
}
```

### Change Icons

Replace SVG paths with your preferred icons from:
- [Heroicons](https://heroicons.com/)
- [Feather Icons](https://feathericons.com/)
- [Font Awesome](https://fontawesome.com/)

### Adjust Spacing

Modify padding and margin values:

```css
.container {
    padding: 2rem 1.5rem; /* Adjust as needed */
}
```
